# **App Name**: RideSelect

## Core Features:

- Dynamic Motorcycle Catalog: Browse and filter the full Honda lineup by category, engine capacity, and OTR pricing with a modern grid interface.
- Real-time Credit Calculator: Interactive slider-based tool for users to adjust Down Payment and tenure to see instant monthly installment updates.
- AI Buying Assistant: An intelligent purchase consultant tool that analyzes user needs and budget to recommend the most suitable motor models and credit packages.
- Side-by-Side Comparison Hub: Technical and financial comparison tool for up to three motorcycles to help users visualize spec and pricing differences.
- Integrated Lead Capture System: Seamless credit application submission for processing in the dealer's backend SQL database.
- Digital Spec Sheets: Detailed product pages featuring high-resolution galleries and interactive 360-degree technical specifications.

## Style Guidelines:

- Primary color is a high-precision Engineering Red (#b31933), evoking the brand's heritage of performance.
- Background is a clean, modern off-white (#fdfafb) with a subtle hint of red to keep the aesthetic warm yet technical.
- Accent color is a deep crimson magenta (#7a1f5c), used for interactive elements to create sophisticated visual contrast.
- A refined pairing using 'Space Grotesk' for technical headlines to emphasize engineering, and 'Inter' for clear, objective body text. Note: currently only Google Fonts are supported.
- Line-based, geometric icons that reflect the structural lines of automotive design.
- A spacious, airy grid layout with card-based components and subtle drop shadows to emphasize content depth.
- Fluid micro-interactions when sliding credit inputs and smooth page transitions using framer-motion.
